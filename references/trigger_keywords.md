## Trigger Conditions

The skill should trigger when:
1. User input contains keywords from the lists above AND
2. User input contains text requirements (words like "写", "标题", "text", "title", etc.)

If only keywords are present without text requirements, this might be a general image generation request and should not trigger this skill.

### Text Requirement Keywords by Language

| Language | Text Keywords |
|----------|--------------|
| 简体中文 | 写、标题、文字、标注、说明 |
| 繁體中文 | 寫、標題、文字、標註、說明 |
| 日本語 | 書く、タイトル、テキスト、説明 |
| 한국어 | 쓰기、제목、텍스트、설명 |
| English | write, title, text, caption, label |
